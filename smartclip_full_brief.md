# SmartClip — полный бриф

## Что это

Замена стандартного буфера обмена Windows. Программа висит полностью в фоне — невидимо, как системный буфер. По `Win+V` открывается окно поверх всего. После вставки или клика в пустое место — закрывается. Системный `Win+V` отключается через реестр при установке.

## Интерфейс

### Главное окно

- Открывается по `Win+V`, висит поверх всех окон
- Закрывается после вставки или клика в пустое место
- Acrylic фон — полупрозрачное размытие того что за окном
- Работает на Windows 10 и Windows 11
- Кастомный titlebar без стандартных рамок Windows
- Скруглённые углы

### Структура окна

```
┌─────────────────────────────────────────────────┐
│  Закрепы (папки)  │  Профили биндов             │  ← всегда сверху
├───────────────────┴─────────────────────────────┤
│                                                  │
│  [Плашка текста]    │  [Скриншот]                │
│  [Плашка текста]    │  [Скриншот]                │
│  [Плашка текста]    │  [Скриншот]                │
│  [Плашка текста]    │  [Скриншот]                │
│  ...                │  ...                       │
│                     │                            │
│  СЛЕВА — история    │  СПРАВА — история           │
│  текстов            │  картинок/скринов          │
│                                                  │
├──────────────────────────────────────────────────┤
│  Поиск по истории и закрепам                     │
└──────────────────────────────────────────────────┘
```

### Лимит истории

- Показывается сразу: 30–40 последних записей
- Долистал до низа — подгружается архив (до 150 штук)
- Архив полный + новая запись → самая старая удаляется автоматически
- После перезагрузки ничего не теряется — всё в SQLite и файлах

### Закрепы

- Отображаются сверху всегда
- Папки с именами, внутри тексты и картинки
- Не удаляются по лимиту — только вручную
- Доступны в поиске

---

## Горячие клавиши и профили

- `Win+V` — открыть SmartClip (вместо системного)
- Профили биндов: создаёшь профиль (например "Программирование"), назначаешь конкретный текст из закрепов на комбинацию клавиш — и он вставляется сразу при нажатии
- Профили переключаются прямо в верхней части главного окна
- Управление профилями — тоже прямо в главном окне

---

## Хранение

### Картинки
Сохраняются как `.png` файлы в `AppData\SmartClip\images\`. В SQLite хранится только путь к файлу. База остаётся лёгкой и быстрой.

### База данных — структура таблиц

| Таблица | Содержимое |
|---------|-----------|
| `history` | тип (текст/картинка), содержимое или путь к файлу, дата |
| `pins` | папка, название, содержимое или путь к файлу |
| `profiles` | имя профиля, комбинация клавиш, текст для вставки |

## Перехват Win+V

1. При установке в реестре `HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced` → `DisabledHotkeys` добавляется `V`
2. Windows перестаёт обрабатывать `Win+V` сама
3. SmartClip регистрирует `Win+V` через `RegisterHotKey`
4. При удалении — `V` убирается из реестра, `UnregisterHotKey` вызывается, всё возвращается

---

## Перехват буфера

1. Пользователь нажал `Ctrl+C` где угодно
2. Windows кладёт данные в свой буфер и шлёт `WM_CLIPBOARDUPDATE`
3. `ClipboardManager` ловит событие → `OpenClipboard` → читает данные → сохраняет → `CloseClipboard`

При вставке из SmartClip — данные кладутся обратно в буфер через `SetClipboardData`, дальше обычный `Ctrl+V`.

---

## Надёжность

**Watchdog поток** — отдельный поток раз в несколько секунд проверяет что `ClipboardManager` живой. Если нет — автоматически перезапускает и заново вызывает `AddClipboardFormatListener`. Пользователь ничего не замечает.

---

## Фоновая работа

- Нет иконки в трее
- Нет видимых окон в фоне
- Приложение полностью невидимо пока не нажат `Win+V`
- Добавляется в автозапуск при установке

## Установщик

- Отдельный `Installer.exe` в стиле самого SmartClip (Acrylic UI)
- При запуске сразу запрашивает UAC
- Делает всё автоматически:
  1. Копирует файлы в `Program Files`
  2. Правит реестр — отключает системный `Win+V`
  3. Добавляет SmartClip в автозапуск
- Финальный экран с двумя кнопками:
  - **Перезагрузить сейчас** — `ExitWindowsEx(EWX_REBOOT, 0)`
  - **Позже** — предупреждение что `Win+V` заработает после перезагрузки

---

## Деинсталляция

1. Убирает `V` из `DisabledHotkeys` в реестре
2. Вызывает `UnregisterHotKey`
3. Перезапускает Explorer
4. `Win+V` снова открывает стандартную историю Windows
5. Все файлы, папка с картинками и база данных удаляются

## Синхронизация (v2)

- Вход через Google аккаунт (OAuth)
- Синхронизируются: закрепы + профили биндов
- История — только локально (приватность)
- Технически: Firebase Firestore или Google Drive API через `QNetworkAccessManager`
